# **ParticRatch**
_ver 0.1_beta_

**RPGツクール**や**Wolf RPG Editor**で使われるエフェクト素材を  
Scratchで使用可能な形式(.sb2ファイル)に変換します。  

### インストール方法
1. ParticRatchをダウンロードする。  
<https://github.com/XmadkittenX/ParticRatch/releases>
2. ダウンロードしたzipファイルを好きな所に展開する。
